# OpenCVSubtitlePlayer

Плеер видео на C++ (OpenCV), который отображает субтитры из `.srt` поверх видео и позволяет управлять воспроизведением, смещением субтитров и перемоткой.

## Функциональность

- Воспроизведение видео через OpenCV (`cv::VideoCapture`)
- Загрузка субтитров в формате **SRT**
- Отрисовка субтитров поверх кадра:
  - перенос строк по ширине кадра
  - обводка текста для читаемости
- Progress bar снизу:
  - отображение текущего времени и длительности (mm:ss)
  - прогресс по длительности видео
- Управление с клавиатуры:
  - пауза/плей
  - перемотка назад/вперёд
  - смещение субтитров вперёд/назад

## Управление

- `Space` — пауза/продолжить
- `A` — перемотка назад на 5 секунд
- `D` — перемотка вперёд на 5 секунд
- `J` — сдвиг субтитров **назад** на 100 мс
- `K` — сдвиг субтитров **вперёд** на 100 мс
- `0` — сброс смещения субтитров в 0
- `Q` или `Esc` — выход

## Сборка проекта

Требуется:
- C++20 compiler (g++/clang++)
- CMake
- OpenCV 4 (установка в системе, pkg-config)

Проект собирается с использованием **CMake** и **OpenCV**.

```bash
cmake -S . -B build
cmake --build build -j8S
```
## Запуск

### Явное указание видео и субтитров
```bash
./build/player video.mp4 subtitles.srt
```
### Автопоиск субтитров рядом с видео

Если второй аргумент не указан, программа пытается найти файл субтитров
в той же директории, где находится видео.
